# SPOR

Spor tarihi ve spor kültürü çalışmaları için dijital arşiv ve çalışma alanı.

## Amaç
- Araştırma notları
- Kaynak listeleri
- Veri setleri
- Analiz ve çıktı dosyaları

## Önerilen klasör yapısı
```
SPOR/
├─ docs/          # genel dokümantasyon
├─ articles/      # makale taslakları
├─ sources/       # bibliyografya, kaynaklar
├─ datasets/
│  ├─ raw/        # ham veri
│  └─ processed/  # temizlenmiş veri
└─ README.md
```

## Yol haritası
1. README ve .gitignore
2. sources/README.md içinde kaynak listesi
3. datasets/ klasörü ve ilk veri setleri
